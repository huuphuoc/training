Compiling LESS to CSS

We can compile *.less to *.css with:

  1. Online tools
  2. Javascript
  3. Programming languages
  4. Separated software
  5. IDE

Online tools

less2css
  • Free, open source
  • Live preview
  • Show errors
  • Easy to use
  • Allow to choose LESS versions
  • Cannot download CSS file

Online tools

lesstester
  • Free, open source
  • LESS cheat sheet
  • No live preview
  • Unable to choose LESS versions
  • Cannot download CSS file
  • Advertising banner

Javascript

less.js

<link rel="stylesheet/less" type="text/css" href="styles.less" />
<script type="text/javascript">
// Options (not required)
less = {
    env: 'development',
    async: false,
    fileAsync: false,
    poll: 1000,
    functions: {},
    dumpLineNumbers: 'comments',
    relativeUrls: false,
    rootpath: ':/a.com/'
};
</script>
<script src="less.js" type="text/javascript"></script>

less.js
  • Run on every browsers
  • Do not need any additional software
  • Written by LESS author
  • Compile every time

Programming languages

lessphp

<?php
require "lessc.inc.php";

$less = new lessc;
echo $less->compile(".block { padding: 3 + 4px }");
echo $less->compileFile("input.less");
